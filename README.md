# ssadkov.github.io

One-page personal website (static) for GitHub Pages.

## Files

- `index.html` — page content
- `styles.css` — styles
- `assets/portfolio.pdf` — portfolio PDF (embedded + downloadable)

## GitHub Pages

Create a public repository named `ssadkov.github.io`, push these files to `main`,
then enable Pages in **Settings → Pages** with:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**
